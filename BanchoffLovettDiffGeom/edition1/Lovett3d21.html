This demo includes four graphical display windows.

The window "Curve: X(t)" displays the curve X(t), the point X(t0), the unit tangent, unit normal, and unit binormal vectors T(t0), P(t0), and B(t0), and the osculating, normal, and rectifying planes.

The other three windows show the projections of X(t) into the osculating, normal, and rectifying planes and the vectors used to define those planes.

You can change t0 and X(t) in the control window. The control window includes a list of functions which you can use for X(t), along with parameters used to define those functions.
