The ability to add and remove features of a given demo is a particularly nice feature of these demos. We show how to add features by starting with a ''blank'' demo, which that it has no predefined elements.

Suppose that we wish to plot the graph of a function f(t)=t3 for t in the interval [0,3]. We would take the following steps:

  1. Under the Controls menu we would select the option Add New Interval. Following the instructions given for the format, we would type in t=0,3,20, where we have chosen 20 as a number of subintervals. (This simply controls how smoothly we sample values from that interval.) Click OK.
  2. Next, under the Controls menu, we would select the option Add New Function. For the proposed function, we would simply type in f(t)=t^3 into the textarea. Click OK.
  3. Under the Controls menu, we would then select the New 2D Graph. Immediately another window appears, but it is empty.
  4. In this new window, we would select Plot/Add Plot/Curve.
  5. Following the instructions in this panel, we would first type in a parametric curve that we want. For our example, we would type in (t,f(t)).
  6. Within the curve definition panel, one has many options for how to color the curve, including coloring by a gradient of some other function. We leave the interested reader to explore these options. Click OK.

Now that the demo has been created (modified) one can change the interval for t or the function f(t) in the main Demo control window.
