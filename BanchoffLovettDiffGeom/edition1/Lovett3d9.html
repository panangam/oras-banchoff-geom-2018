This demo includes two graphical display windows.

The window "Parametric Curve and Velocity Vector" displays the curve X(t), the point X(t0), and the velocity vector X'(t0) with its tail at X(t0).

The window "Velocity Curve and Acceleration Vector" displays the curve X'(t), the point X'(t0), and the acceleration vector X''(t0) with its tail at X(t0).

You can change t0 and X(t) and choose whether or not to show the entire curve, the position vector X(t0), and the acceleration vector X''(t0), in the control window. The control window includes a list of functions which you can use for X(t), along with parameters used to define those functions.
