This demo displays the curve X(t) = (xdisp + (1 + 0.5cos(t + π/4))cos(2t), (1+ 0.5cos(t + π/4))sin(2t), cos(3t)), the point X(t0), and the z-axis in green. The control window includes a readout which displays the winding number of X(t) about the z-axis. It is possible to change t0, xdisp, and X(t) and to choose whether or not to show the entire curve and the position vector X(t0) in the control window. Observe what happens to W for the default curve as xdisp runs from 0 to 2. What happens to the curve X(t) when W changes?
