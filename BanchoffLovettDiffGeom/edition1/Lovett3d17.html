This demo includes four graphical display windows.

The window "Curve: X(t)" displays the curve X(t), the point X(t0), the acceleration vector X''(t0) (the diagonal of the yellow rectangle, shown in pink), and the tangential and normal components of acceleration (shown in magenta and green, respectively).

The windows "Total Acceleration", "Tangential Component", and "Normal Component" display the magnitude of the acceleration vector |X''(t0)|, the magnitude of the tangential component of acceleration, and the magnitude of the normal component of acceleration, respectively.

You can change t0 and X(t) in the control window. The control window includes a list of functions which you can use for X(t), along with parameters used to define those functions.
