This demo includes two windows.

The window "Curve: X(t)" displays the curve X(t) defined in the conrol window and the point X(t0).

The window "Perturbed Tangential Image" displays the tangent vector T(t0), a curve that it traces out, and an arc showing the direction angle. The tangent vector is perturbed slightly (with the magnitude of this perturbation controlled by parameter u) so that the curve that it traces out does not overlap itself.

You can change t0 and u in the control window.
