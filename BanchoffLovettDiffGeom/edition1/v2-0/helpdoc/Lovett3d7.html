This demo includes a single graphical window that displays the curve X(t), the point X(t0), the position vector X(t0), and the tangent vector T(t0) with its tail at X(t0).

You can change t0 and X(t) in the control window. The control window includes a list of functions that you can use for X(t), along with parameters used to define those functions.

As in previous demos is a "Show Color Spectrum" checkbox.
