This demo includes two graphical display windows.

The window "Curve: X(t)" displays the curve X(t), the point X(t0), and unit tangent, principal normal, and binormal vectors T(t0), P(t0), and B(t0), with their tails at X(t0).

The window "Unit Tangent (T), Unit Normal (P), and Unit Binormal (B)" displays the unit tangent, principal normal, and binormal vectors T(t0), P(t0), and B(t0) and the unit sphere on which their heads are constrained to lie.

You can change t0 and X(t) in the control window. The control window includes a list of functions that you can use for X(t), along with parameters used to define those functions.

As in previous demos there is a "Show Color Spectrum" checkbox.
