The window "Curve: X(t)" displays the curve X(t) = (cos(t), sin(t), cos(2t)/c), the point X(t0), and the Frenet Frame of the curve.

The other three window display the three indicatrices on a transparent sphere.

As in earlier demos, there is a "Show Color Spectrum" checkbox. Additionally, the "Show Entire Indicatrices" checkbox displays the complete path of each vector in the Frenet Frame.
